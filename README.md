# rc-steps-image

---

React steps component made to work with images instead of text

## Development

```
npm install
npm start
```

## Usage

CHECK THE ONLY EXAMPLE PROVIDED TO SEE HOW TO USE IT!

```jsx
<Steps current={1}>
  <Steps.Step title="first" />
  <Steps.Step title="second" />
  <Steps.Step title="third" />
</Steps>
```
